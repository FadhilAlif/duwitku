# Assets Folder

## Logo

Letakkan file logo aplikasi Anda di sini dengan nama `logo.png`

Anda dapat menggunakan logo yang sudah Anda miliki dari gambar yang dikirimkan (Duwitku logo dengan "OJO BOROS!" tagline).

### Rekomendasi:

- Format: PNG dengan background transparan
- Ukuran: 512x512 px atau lebih besar
- Nama file: `logo.png`

Untuk sementara, AppLogo widget akan mencoba memuat `assets/logo.png`. Pastikan file tersebut ada sebelum menjalankan aplikasi.
